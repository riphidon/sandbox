# sandbox-api
Simple rest API sandbox.
Plugged to work with a postgreSQL database.
Sample .config file : 
`{
  "port": 3000,
  "env": "dev",
  "pepper": "alaf-lamout",
  "hmac_key": "secret-hmac-key",
  "database": {
    "host": "host_name",
    "port": 0000,
    "user": "user",
    "password": "password",
    "name": "db_name"
  }
}
`
